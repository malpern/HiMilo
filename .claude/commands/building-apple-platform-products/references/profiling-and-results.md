# Profiling and Test Results

Performance tracing with xctrace and test result extraction with xcresulttool.

**Important**: `xcrun xctrace list devices` outputs to **stderr**, not stdout. When parsing in scripts, redirect with `2>&1`.

## xctrace (Replaces instruments)

The `instruments` CLI was removed in Xcode 13. Use `xcrun xctrace` instead.

### Listing Devices and Templates

```bash
# List available devices (note: outputs to stderr)
xcrun xctrace list devices 2>&1

# List available profiling templates
xcrun xctrace list templates
```

### Recording Traces

```bash
# Profile a running app
xcrun xctrace record --template "Time Profiler" --output profile.trace --attach MyApp

# Profile with time limit
xcrun xctrace record --template "Time Profiler" --output profile.trace --time-limit 30s --attach MyApp

# Profile on a specific device
xcrun xctrace record --template "Time Profiler" --device <UDID> --output trace.trace --attach MyApp
```

### Exporting Trace Data

```bash
# Export table of contents
xcrun xctrace export --input profile.trace --toc
```

## xcresulttool

Extract data from `.xcresult` bundles generated by xcodebuild test.

### Test Results

```bash
# Get test summary
xcrun xcresulttool get test-results summary --path TestResults.xcresult

# Get individual test results
xcrun xcresulttool get test-results tests --path TestResults.xcresult

# Export result bundle contents
xcrun xcresulttool export --path TestResults.xcresult --output-path ./exports --type directory
```

### Generating Result Bundles

```bash
xcodebuild test -scheme "MyApp" \
  -destination 'platform=iOS Simulator,name=iPhone 17' \
  -resultBundlePath ./TestResults.xcresult
```

## Code Coverage (xccov)

```bash
# Summary report
xcrun xccov view --report TestResults.xcresult

# JSON format
xcrun xccov view --report --json TestResults.xcresult

# File-level detail
xcrun xccov view --file-list TestResults.xcresult
```

## Quick Reference

| Goal | Command |
|------|---------|
| List devices | `xcrun xctrace list devices 2>&1` |
| List templates | `xcrun xctrace list templates` |
| Record trace | `xcrun xctrace record --template "<name>" --output <file> --attach <app>` |
| Export trace | `xcrun xctrace export --input <file> --toc` |
| Test summary | `xcrun xcresulttool get test-results summary --path <xcresult>` |
| Test details | `xcrun xcresulttool get test-results tests --path <xcresult>` |
| Coverage report | `xcrun xccov view --report <xcresult>` |
| Coverage (JSON) | `xcrun xccov view --report --json <xcresult>` |

## Troubleshooting

### xctrace output not captured in scripts

**Cause**: `xcrun xctrace list devices` outputs to stderr, not stdout.

**Solution**:
```bash
xcrun xctrace list devices 2>&1
```

### "instruments" command not found

**Cause**: `instruments` was removed in Xcode 13.

**Solution**: Use `xcrun xctrace` instead. See the recording section above for equivalent commands.
