VoxClaw Help › Codex Setup

Connect to Codex

VoxClaw integrates with OpenAI's Codex CLI using the same hook system. Every Codex response is spoken aloud, and sending a new message automatically stops the current reading.

Install

Run the setup script in your terminal:

git clone https://github.com/malpern/VoxClaw.git
bash VoxClaw/plugins/voxclaw/setup-codex.sh

This copies hook scripts to ~/.codex/hooks/ and adds entries to ~/.codex/config.toml.

What Gets Added

The setup script adds two hook entries to your Codex config:

[[hooks.Stop]]
hooks = [
  { type = "command", command = "~/.codex/hooks/voxclaw-codex-stop-speak.sh", timeout = 5000 }
]

[[hooks.UserPromptSubmit]]
hooks = [
  { type = "command", command = "~/.codex/hooks/voxclaw-codex-ack.sh", timeout = 2000 }
]

How It Works

Codex provides the last assistant message directly in the Stop hook payload, so VoxClaw can speak responses without parsing the full transcript. The hook strips markdown formatting, shortens URLs, and preserves paragraph breaks for natural speech pacing.

Verify It Works

Start a new Codex session and ask it something. You should hear the response spoken aloud with the teleprompter overlay.

Running Codex sessions need to be restarted to load newly installed hooks.
