Voice Engines

VoxClaw supports three voice engines. You can switch between them at any time in Settings.

Apple Built-in (Free)

Uses your Mac's built-in text-to-speech. All processing happens on-device — no internet required, no API key needed. This is the default.

Quality varies by voice. The "Siri" voices sound the most natural.

OpenAI ($)

Uses OpenAI's gpt-4o-mini-tts model for natural, expressive speech. Requires your own API key from platform.openai.com.

Cost: approximately $0.15 per 10,000 characters. Available voices: Alloy, Ash, Coral, Echo, Fable, Nova, Onyx, Sage, Shimmer.

ElevenLabs ($$)

Premium voice synthesis with the most natural results. Requires your own API key from elevenlabs.io.

Standard: ~$1.80 per 10K chars. Turbo mode: ~$0.60 per 10K chars (faster, lower quality).

Adding an API Key

  1. Open VoxClaw Settings (click menu bar icon › Settings)
  2. In the Voice section, select OpenAI or ElevenLabs
  3. Paste your API key when prompted

Your API key is stored securely in your Mac's Keychain and is never sent to VoxClaw or any server other than the voice provider you chose.

Multi-Agent Voices

When multiple coding agents are sending responses, VoxClaw automatically assigns each project a distinct voice. This uses a deterministic hash of the project directory, so the same project always gets the same voice across sessions.

Speed Control

Adjust speech speed from 0.5x to 2.0x using the Speed slider in Settings. This works with all voice engines.
