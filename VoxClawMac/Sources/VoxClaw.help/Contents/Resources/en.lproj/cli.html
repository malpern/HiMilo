CLI & Integrations

Beyond the Claude Code and Codex plugins, VoxClaw offers several ways to send text for speech.

Command Line

voxclaw "Hello, world!"              # Read text aloud
voxclaw --clipboard                  # Read from clipboard
voxclaw --file article.txt           # Read from file
echo "Hello" | voxclaw              # Read from stdin
voxclaw --voice nova "Hello"         # OpenAI voice override
voxclaw --rate 1.5 "Hello"          # 1.5x speech speed
voxclaw -a "Hello"                   # Audio only (no overlay)

URL Scheme

Trigger VoxClaw from any app that supports URL schemes:

open "voxclaw://read?text=Hello%20world"

HTTP API

Send text to the local listener:

curl -X POST http://localhost:4140/read \
  -d '{"text": "Hello from any tool"}'

See Network & Remote for full API details.

macOS Services Menu

Select text in any app, right-click, and choose Services › Read with VoxClaw.

Shortcuts

Use the Shortcuts app to create automations:

shortcuts run "Read with VoxClaw" --input-string "Hello"

Save Audio to File

With OpenAI voices, save the audio instead of playing it:

voxclaw --output hello.mp3 "Hello world"
