Getting Started

VoxClaw sits in your menu bar, waiting for your coding agent to send text. When it does, VoxClaw speaks it aloud with a floating teleprompter overlay.

Quick Setup

  1. Launch VoxClaw. It appears in your menu bar as a small icon.
  2. Connect your agent. Install the VoxClaw plugin for Claude Code or Codex.
  3. Choose a voice. Works immediately with your Mac's built-in voice. For natural voices, add an OpenAI or ElevenLabs API key in Settings.

How It Works

VoxClaw runs a local HTTP listener on port 4140. When your agent finishes a response, the plugin sends the text to this listener. VoxClaw speaks it aloud and shows the teleprompter overlay with synchronized word highlighting.

If you start typing before VoxClaw finishes reading, it stops automatically.

Menu Bar Controls

  • Pause / Resume — pause current speech
  • Stop — stop playback entirely
  • Read Clipboard — speak whatever is on your clipboard
  • Settings — voice, overlay, and integrations
Tip: VoxClaw uses Apple's text-to-speech by default — no API key needed, everything processed on-device.
