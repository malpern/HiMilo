Getting Started

VoxClaw sits in your menu bar, waiting for your coding agent to send text. When it does, VoxClaw speaks it aloud with a floating teleprompter overlay that highlights each word in sync.

Quick Setup

  1. Launch VoxClaw. It appears in your menu bar as a small icon.
  2. Connect your agent. Install the VoxClaw plugin for Claude Code or Codex. Your agent will automatically speak its responses through VoxClaw.
  3. Choose a voice. VoxClaw works immediately with your Mac's built-in voice (free, no setup). For more natural voices, add an OpenAI or ElevenLabs API key in Settings.

How It Works

VoxClaw runs a local HTTP listener on port 4140. When your agent finishes a response, the VoxClaw plugin sends the text to this listener. VoxClaw then speaks it aloud and shows the teleprompter overlay.

If you start typing your next message before VoxClaw finishes reading, it stops automatically — no interruption needed.

Menu Bar Controls

Click the VoxClaw icon in your menu bar to:

  • Pause / Resume current speech
  • Stop playback entirely
  • Read Clipboard — speak whatever is on your clipboard
  • Settings — configure voice, overlay appearance, and integrations
VoxClaw uses Apple's text-to-speech by default, which requires no API key and processes everything on your device.
