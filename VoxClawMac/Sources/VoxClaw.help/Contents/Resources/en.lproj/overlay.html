Teleprompter Overlay

The floating overlay shows the text being spoken with synchronized word highlighting. It appears automatically when speech starts and fades away when it finishes.

Features

  • Word highlighting — the current word is highlighted as it's spoken
  • Project indicators — colored badges show which agent/project is speaking
  • Code formatting — inline code appears in a distinct color
  • Paragraph breaks — text flows naturally with paragraph spacing
  • Content fade — smooth transitions between queued responses

Customization

Open Settings › Overlay Appearance to customize:

  • Style presets — choose from built-in appearance styles
  • Colors and opacity — adjust to match your workspace
  • Position — drag the overlay to reposition it. Enable "Remember overlay position" in Settings to persist the location.

Audio Only Mode

If you prefer to hear responses without the visual overlay, enable "Audio only" in Settings. VoxClaw will still speak responses but won't show the floating panel.

Non-Activating

The overlay never steals focus from your current application. It floats above other windows but you can continue typing and working while it displays.
