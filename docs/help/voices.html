Voice Engines

Three engines — switch anytime in Settings.

Apple Built-in (Free)

On-device. No internet, no API key needed. This is the default.

OpenAI ($)

Natural speech via gpt-4o-mini-tts. ~$0.15 per 10K characters. Bring your own API key.

ElevenLabs ($$)

Premium synthesis. Standard ~$1.80 / Turbo ~$0.60 per 10K chars. Bring your own key.

API Keys

Open Settings, select your engine, paste the key. Stored in your Mac's Keychain — never sent to VoxClaw.

Per-Project Voices

Multiple agents get distinct voices via deterministic hashing. Same project, same voice, every time.
