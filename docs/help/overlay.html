Teleprompter Overlay

A floating overlay with synchronized word highlighting.

  • Word highlighting — current word highlights as spoken
  • Project badges — shows which agent is speaking
  • Code color — inline code in a distinct color
  • Non-activating — never steals focus

Customize

Settings offers style presets, font, colors, opacity, and padding. Enable "Remember overlay position" to keep it where you put it.

Audio Only

Toggle "Audio only" in Settings to skip the overlay entirely.
