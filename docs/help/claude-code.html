Claude Code Setup

Make every response spoken aloud automatically.

Install via Marketplace

claude plugin marketplace add malpern/VoxClaw
claude plugin install voxclaw

Manual Install

git clone https://github.com/malpern/VoxClaw.git
bash VoxClaw/plugins/voxclaw/setup-claude-code.sh

What It Does

Stop Hook

Extracts the response, strips markdown, sends to VoxClaw to speak.

UserPromptSubmit Hook

Tells VoxClaw to stop reading when you send a new message.

Multi-Agent Voices

Multiple sessions in different directories each get a distinct voice. Responses queue — one speaks at a time.

Note: Restart running sessions to load new hooks.
