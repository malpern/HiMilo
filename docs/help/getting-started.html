Getting Started

VoxClaw sits in your menu bar, waiting for your coding agent to send text.

Quick Setup

  1. Launch VoxClaw. It appears in your menu bar.
  2. Connect your agent. Install the plugin for Claude Code or Codex.
  3. Choose a voice. Works with your Mac's built-in voice. Add an OpenAI or ElevenLabs key for natural voices.

How It Works

VoxClaw runs a local HTTP listener on port 4140. When your agent finishes a response, the plugin sends text to this listener. VoxClaw speaks it aloud with a synchronized teleprompter overlay.

Start typing before it finishes and it stops automatically.

Menu Bar

  • Pause / Resume current speech
  • Stop playback
  • Read Clipboard — speak clipboard contents
  • Settings — voice, overlay, integrations
Tip: Apple's built-in voice works with no API key — everything on-device.
